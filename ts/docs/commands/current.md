# wallet-cli current

Show the current (active) account.

## Synopsis

```
wallet-cli current [options]
```

## Options

[Global options](index.md) only.

## Exit status

`0` success · `1` execution failure · `2` usage error. See [machine-interface](../machine-interface.md).
