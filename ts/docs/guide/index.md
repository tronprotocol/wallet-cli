# Guides

Task-oriented walkthroughs. New here? Start with **Getting started**, then reach for the others as you need them. Every example runs on the Nile testnet.

| Guide | What it covers |
|---|---|
| [Getting started](getting-started.md) | Create a wallet, fund it on Nile, check the balance, send your first transaction |
| [Sending tokens](send-tokens.md) | Send native TRX and TRC20 / TRC10 tokens; the token address book |
| [Staking and resources](stake-and-resources.md) | Stake, delegate, and unstake TRX to earn energy / bandwidth |
| [Using a Ledger](ledger.md) | Register and sign with a Ledger hardware wallet |
| [Scripting](scripting.md) | Call wallet-cli from shell scripts and CI, safely |

For per-command detail see the [command reference](../commands/index.md); for the JSON contract see [machine-interface](../machine-interface.md).
